# 33_多媒体播控系统 (精简版)

本项目是原 Next.js 版本的重构精简版，旨在提供一个**简洁、明了、易修改**的单页网站。

## 1. 项目特点
- **无运行时**：纯 HTML5 + Tailwind CSS，不依赖 React/Next.js，加载速度极快。
- **单文件维护**：所有的内容、样式和基础逻辑都在 `web/index.html` 中，修改文案就像编辑文档一样简单。
- **响应式设计**：完美适配移动端、平板和桌面端。
- **Bilibili 视频支持**：集成了展馆播控系统的演示视频。

## 2. 目录结构
- `web/index.html`：网站核心文件。
- `docs/articles/`：预留的技术解决方案文章。
